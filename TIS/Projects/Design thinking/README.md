  <h1>Automatic Air Conditioner Schedule</h1>
  <p>A design thinking project developed by Group <strong>Santai Pro Max</strong> to improve energy efficiency in campus classrooms through smart HVAC management.</p>
  <p>Youtube Video : https://www.youtube.com/watch?v=3UUuz9on-7A</p>

  <hr>

  <h2>Design Thinking Methodology</h2>
  <p>This project implemented the five core stages of design thinking to solve energy wastage:</p>
  <ul>
    <li>
      <strong>Empathize:</strong> Gathered student data via Google Forms and interviews to understand energy efficiency issues on campus.
    </li>
    <li>
      <strong>Define:</strong> Identified that AC units are often set to fixed, overcooled temperatures and left running in empty classrooms.
    </li>
    <li>
      <strong>Ideate:</strong> Brainstormed an automated system using motion and temperature sensors to adjust AC settings based on room occupancy.
    </li>
    <li>
      <strong>Prototype:</strong> Developed a visual model featuring a user-friendly app for classroom temperature monitoring and control.
    </li>
    <li>
      <strong>Test:</strong> Evaluated the prototype's operational capacity and features to ensure effectiveness and user satisfaction.
    </li>
  </ul>

  



  <h2>Key Features & Solutions</h2>
  <ul>
    <li><strong>Motion Detection:</strong> Automatically turns off or adjusts the AC when no movement is detected for a set duration.</li>
    <li><strong>Standby Mode:</strong> Maintains a consistent temperature (e.g., 24Â°C) instead of frequent power cycles to prevent high energy consumption.</li>
    <li><strong>Mobile Integration:</strong> A user-friendly app that displays real-time settings and allows manual overrides.</li>
  </ul>
